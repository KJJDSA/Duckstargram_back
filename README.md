## ✨ 사용한 Backend 기술 스택

- express 프레임워크를 기반으로 한 Rest API 서버
- MySQL(sequelize) 관계형 데이터베이스를 활용하여 유저 정보 저장
- jsonwebtoken login 방식 구현
- bcrypt 로 유저 암호 해쉬화
- socket.io 찍먹, 채팅 시스템 구현
